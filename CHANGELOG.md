## [0.2.0] — 2026-06-30

### Added
- **교사 입장 개편**: 세션 생성 시 반 이름·교사 이름 입력 필드 추가. 코드 자동 생성.
- **이 기기에서 들어갔던 반 목록**: 최근 접속한 반을 반 이름·교사 이름 포함해 로컬 목록으로 표시. 목록에서 클릭하면 바로 입장.
- **코드로 입장 시 반 정보 저장**: 코드 직접 입력으로 입장해도 반 이름·교사 이름을 DB에서 읽어 목록에 자동 추가.
- **교사 콘솔 헤더**: 입장 후 헤더에 반 이름·교사 이름 표시.
- **슈퍼어드민 콘솔** (`/admin`): 별도 암호로 보호되는 관리자 전용 화면. 전체 세션을 실시간으로 모니터링. 반 이름·교사 이름·단계·접속 학생·작품 수·감상 답안 수 표시. 코드 검색(반 이름 포함), 교사 콘솔 바로 열기·TV·QR 링크 제공.
- **RTDB 보안 규칙** `adminAuth` 경로 추가 (슈퍼어드민 암호 write-once).
- **작품 비공개 옵션** (`🔒 공유 안 함`): 작품 카드에 체크박스 추가. 체크된 작품은 다른 반의 가져오기 목록에서 제외됨.
- **다른 반 작품 가져오기 개편**:
  - 반 선택 없이 모든 반의 공유 작품을 한번에 탐색.
  - 공통감상 / 선택감상 / 경매대상 탭으로 분류.
  - 썸네일·제목·출처 반 이름 포함 체크박스 목록으로 개별 선택.
  - 탭별 전체 선택/해제, 탭 넘나들며 합산 선택 가능.
  - 선택 후 가져올 위치(공통감상실 / 선택감상실) 라디오로 지정 후 가져오기.
- **`/teacher?code=XXX` 딥링크**: 슈퍼어드민 "열기" 버튼에서 코드 파라미터로 교사 콘솔 바로 진입.
- **홈 화면** 슈퍼어드민 진입 링크 추가 (하단 작은 텍스트 버튼).

### Changed
- `SessionMeta`에 `className`, `teacherName` 필드 추가.
- `RecentSession`에 `className`, `teacherName` 필드 추가.
- `Artwork`에 `isPrivate` 필드 추가 (선택적, 기본 공개).
- `copyArtworksFrom` → `copySelectedArtworks`로 교체 (작품 배열 직접 전달 + 배치 덮어쓰기 지원).
- `paths.sessionsRoot()` 헬퍼 추가 (`'sessions'` 루트 경로).
- 다른 반 드롭다운 작품 수가 비공개 제외 "공유 가능" 수로 표시됨.

### Fixed
- 슈퍼어드민 검색이 코드뿐 아니라 반 이름도 포함해 필터링.

## [0.1.0] — 2026-06-27

### Added
MVP 최초 릴리스. PRD v1.1 기준.

- 교사 콘솔: 세션 생성(코드 발급), 단계 전환, 작품 업로드(URL·이미지 서버), 감상 발문/프롤로그 편집, 모둠 설정, 현황판(학생별 감상 진행도), 결과 발표, CSV 내보내기.
- 학생 태블릿: 코드+번호+이름 입장, 프롤로그, 공통작품감상실(수행평가), 선택작품감상실, 경매장(모둠 대표 입찰), 결과 확인.
- TV 송출 화면: 경매 현황 실시간 표시.
- Firebase RTDB 실시간 동기화, 익명 인증, 보안 규칙.
- Cloudflare Pages 배포 (GitHub Actions CI).
- 다크 미술관 + 골드 액센트 디자인 시스템.
- 경매 3가지 방식: 호가 주도 / 입찰(sealed) / 수동 기록.
