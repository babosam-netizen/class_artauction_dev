## Step 4 — 기능별 (각 기능 통과 시 다음 기능)

### 4a. 감상 + 해설
- [ ] 작품별 발문(가변 개수) 패널, 한 번에 하나씩 + 자유 텍스트 저장 → `appreciations` 기록
- [ ] 제출 후 해설 화면 표시 → 확인 후 다음 작품
- [ ] 진행 도트/화살표로 본 작품 재열람
- [ ] 현재 열린 단계 작품만 수정 가능, 이전 단계는 읽기 전용
- [ ] 수행평가 대상(공통회랑)만 평가 표식, 분기 감상은 저장하되 비평가

### 4b. 회랑 분기
- [ ] 교사 문 개수 설정, 문별 작품 배치
- [ ] 학생 문 선택 입장 → 문별 작품 감상 저장
- [ ] 모둠원이 다른 문 선택 유도(안내)

### 4c. 경매 + 타이머
- [ ] 모둠 대표 1명만 입찰(대표 기기 게이트)
- [ ] 최소 호가 단위 강제, 현재가+단위 이상만 유효
- [ ] 모둠 예산 상한(시작자금−지불가합) 초과 입찰 차단/경고
- [ ] 작품 랜덤 순서, 교사가 다음 작품 진행
- [ ] 교사 타이머: 호가 들어오면 자동 연장, 무호가 시 자동 낙찰
- [ ] 유찰 처리 + 유찰작 재경매
- [ ] **RTDB 트랜잭션으로 최고가 원자적 갱신**(동시 입찰 동시성 테스트 통과)
- [ ] TV·학생 화면에 현재가·입찰모둠·타이머·로그 실시간 반영

### 4d. 결과 + 자산 순위
- [ ] 자산 = 잔여현금 + 낙찰작 감정가합, 모둠 순위 산출
- [ ] 감정가·출처는 결과 단계에서만 공개(이전엔 비표시)
- [ ] 모둠별 "지불가 vs 감정가" 비교 시각화

### 4e. 데이터 내보내기
- [ ] 학생별 감상(작품·발문·답변) CSV/PDF, 이름·번호 정렬
- [ ] 모둠별 경매결과(낙찰작·가격·잔액·자산·순위) CSV/PDF

### 4f. 프롤로그
- [ ] 학년군 3단계 서사, 단계별 "건너뛰기"
- [ ] 교사 편집 가능(세션 content에 저장)
